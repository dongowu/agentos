# AgentOS v0.1.0 开源首发说明

- 日期：2026-03-09
- 版本：`v0.1.0`
- 面向对象：开发者、平台团队、早期采用者

## 概要

AgentOS v0.1.0 是开源核心仓库的第一次公开发布。

这次发布有意将自己定位为一个**社区基础设施底座**，而不是一个已经完全打磨好的终端产品。当前公开的重点是执行底座本身：控制面、Worker 运行时、调度、审计、遥测，以及围绕这些能力的扩展接口。

## 本次开源包含什么

Community 核心目前已经包含：

- 基于 Go 的控制面，包含 `apiserver`、`controller` 和 CLI 入口
- 基于 Rust 的 Worker Runtime，用于命令执行
- 本地调度与 NATS 调度两条路径
- 任务提交、规划、执行、回放与审计 API
- 任务级与动作级 SSE 遥测流
- agent loop / tool-calling 能力
- 仓库级许可证、安全与贡献文档

## 这次发布适合谁

这次开源更适合以下团队：

- 想 self-host 一个 Agent 执行底座的团队
- 想看清楚执行、调度、审计如何打通的平台工程团队
- 想把 Agent 从脚本提升为“可治理工作负载”的团队
- 想在核心底座之上扩展适配器、技能或垂直工作流的开发者

## 这次发布不是什么

这次发布还不是：

- 完整的企业控制台产品
- 托管云服务
- 完整的组织治理平台
- 打包好的行业工作流产品

这些方向未来可以继续发展，但当前公开仓库的重点仍然是 **Community 核心**。

## 验证方式

本次版本发布前通过了：

- `go test ./... -count=1`
- `./scripts/acceptance.sh`

## 对外口径建议

`v0.1.0` 的对外表达建议保持简单：

> AgentOS 是一个开源的 Agent 执行平台，采用 Go 控制面与 Rust 运行时平面。

这样的表达最符合项目当前实际能力，也最有利于建立技术可信度。
